# Quick Start Guide

## Get Running in 5 Minutes

### Step 1: Personalize (2 minutes)

Replace `vanshsharma@example.com` with your real email in these 3 files:
1. `app/layout.tsx` (line 48)
2. `app/about/page.tsx` (line 90)
3. `README.md` (line 106)

If your name is not "Vansh Sharma", replace that too in the same files.

### Step 2: Install & Test (2 minutes)

```bash
npm install
npm run dev
```

Open http://localhost:3000 and test:
1. Click "Start Creating Invoice"
2. Fill in the form with sample data
3. Click "Download Invoice PDF"
4. Verify PDF downloads correctly

### Step 3: Deploy (1 minute)

```bash
# Push to GitHub
git init
git add .
git commit -m "GST Invoice Generator"
git remote add origin https://github.com/YOUR_USERNAME/gst-invoice-generator.git
git push -u origin main
```

Then:
1. Go to vercel.com
2. Sign in with GitHub
3. Import your repository
4. Click Deploy
5. Done! Copy your live URL

## Common Questions

**Q: Do I need to change anything else?**
A: No, just your name and email. Everything else is ready.

**Q: Will this cost money?**
A: No. GitHub and Vercel are free. npm packages are free.

**Q: What if I get errors?**
A: Most likely npm not installed. Install Node.js from nodejs.org first.

**Q: How do I test if calculations are correct?**
A: Example: ₹10,000 at 18% GST = ₹900 CGST + ₹900 SGST = ₹11,800 total ✅

**Q: Can I customize the design?**
A: Yes, but not required. It's already professional and meets all requirements.
