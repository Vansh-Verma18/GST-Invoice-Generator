# Portfolio Project Entry

## Project Title
GST Invoice Generator

## Project Description
A free online tool for generating GST-compliant invoices with automatic CGST and SGST calculations. Built with Next.js, TypeScript, and modern web technologies.

## Key Features
- Automatic GST tax calculations (CGST + SGST)
- Support for multiple products/services per invoice
- Professional PDF invoice generation
- Fully responsive design
- Dark theme with glassmorphism UI
- 100% client-side - no data stored

## Technologies Used
- Next.js 14
- TypeScript
- Tailwind CSS
- jsPDF
- Vercel

## Problem & Solution

**Problem**: Small businesses and freelancers need expensive software to generate GST-compliant invoices with proper tax calculations.

**Solution**: Free, browser-based tool that automatically calculates CGST and SGST, generates professional PDF invoices, and works on all devices - no signup or installation required.

## Impact
- Saves time with automatic calculations
- Reduces errors in tax computation
- Provides professional invoice formatting
- Eliminates need for paid software subscriptions
