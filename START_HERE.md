## 🧮 How It Works

### GST Calculation Example
```
Product: Web Development
Quantity: 1
Rate: ₹10,000
GST Rate: 18%

Calculations:
Amount = 1 × ₹10,000 = ₹10,000
CGST (9%) = ₹10,000 × 18% ÷ 2 = ₹900
SGST (9%) = ₹10,000 × 18% ÷ 2 = ₹900
Total = ₹10,000 + ₹900 + ₹900 = ₹11,800 ✅
```

## 🆘 Need Help?

### Common Issues

**"npm: command not found"**
→ Install Node.js from nodejs.org

**"Port 3000 already in use"**
→ Run: `npm run dev -- -p 3001`

**"Build failed on Vercel"**
→ Run `npm run build` locally first to see errors

**"PDF not downloading"**
→ Check browser console, try different browser

### Get More Help

1. Check `TESTING_CHECKLIST.md` for debugging
2. Read error messages carefully
3. Search the error on Google
4. Check Next.js docs: nextjs.org/docs
